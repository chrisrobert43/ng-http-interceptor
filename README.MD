# ng2-http-interceptor

> Http Interceptor library for Angular 2

## Installation

To install this library, run:

```bash
$ npm install ng2-http-interceptor --save
```

## Development

To generate all `*.js`, `*.js.map` and `*.d.ts` files:

```bash
$ npm run tsc
```

To lint all `*.ts` files:

```bash
$ npm run lint
```
